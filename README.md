# এডমিশন বাস - বিশ্ববিদ্যালয় ভর্তি পরীক্ষা টিকেটিং প্ল্যাটফর্ম

একটি সম্পূর্ণ প্রোডাকশন-রেডি PHP ভিত্তিক বাস টিকেটিং প্ল্যাটফর্ম যা বাংলাদেশের শিক্ষার্থীদের জন্য তৈরি।

## বৈশিষ্ট্য

### ব্যবহারকারী পক্ষ
- সহজ টিকেট বুকিং সিস্টেম
- একাধিক পেমেন্ট গেটওয়ে (bKash, Nagad, Rocket, কার্ড)
- রিয়েল-টাইম ইনভয়েস ট্র্যাকিং
- তাৎক্ষণিক ইমেইল এবং SMS নোটিফিকেশন
- বহুভাষিক সাপোর্ট (বাংলা/ইংরেজি)

### অ্যাডমিন প্যানেল
- বুকিং ম্যানেজমেন্ট ড্যাশবোর্ড
- রুট এবং সিট ম্যানেজমেন্ট
- পেমেন্ট ট্র্যাকিং এবং সেটেলমেন্ট
- বিস্তারিত অ্যানালিটিক্স এবং রিপোর্ট
- ব্যবহারকারী ব্যবস্থাপনা

### নিরাপত্তা
- SSL/HTTPS এনক্রিপশন
- CSRF প্রোটেকশন
- SQL ইনজেকশন প্রিভেনশন
- XSS প্রোটেকশন
- Rate limiting
- Bcrypt পাসওয়ার্ড হ্যাশিং
- আইপি-বেসড লগিং

## প্রয়োজনীয়তা

- PHP 7.4+
- MySQL 5.7+
- Apache/Nginx
- Composer (ঐচ্ছিক)

## ইনস্টলেশন

1. **ক্লোন করুন বা ডাউনলোড করুন**
   \`\`\`bash
   git clone https://github.com/yourusername/admission-bus.git
   cd admission-bus
   \`\`\`

2. **ডাটাবেস সেটআপ করুন**
   \`\`\`bash
   mysql -u root -p < database/schema.sql
   \`\`\`

3. **কনফিগারেশন সেটআপ করুন**
   \`\`\`bash
   cp config/config.sample.php config/config.php
   \`\`\`
   এবং আপনার ডেটাবেস ক্রেডেনশিয়াল সম্পাদনা করুন।

4. **পেমেন্ট গেটওয়ে সেটআপ করুন**
   - bKash API কী যোগ করুন
   - Nagad মার্চেন্ট আইডি যোগ করুন
   - Rocket API ক্রেডেনশিয়াল যোগ করুন

5. **ওয়েব সার্ভার কনফিগার করুন**
   - `.htaccess` এনেবল করুন
   - HTTPS সক্ষম করুন
   - ফাইল পারমিশন সেট করুন

## ব্যবহার

### হোমপেজ অ্যাক্সেস করুন
\`\`\`
http://yourdomain.com
\`\`\`

### অ্যাডমিন প্যানেল অ্যাক্সেস করুন
\`\`\`
http://yourdomain.com/admin/login.php
\`\`\`

ডিফল্ট ক্রেডেনশিয়াল:
- Username: `admin`
- Password: প্রোডাকশনে পরিবর্তন করুন

## API এন্ডপয়েন্ট

### বুকিং তৈরি করুন
\`\`\`
POST /api/create-booking.php
Parameters: full_name, phone, email, nid, destination, journey_date, num_seats, payment_method
\`\`\`

### পেমেন্ট প্রসেস করুন
\`\`\`
POST /api/process-payment.php
Parameters: booking_id, payment_method, transaction_id, reference
\`\`\`

### ইনভয়েস যাচাই করুন
\`\`\`
POST /api/verify-invoice.php
Parameters: invoice_number
\`\`\`

### রুট পান
\`\`\`
GET /api/get-routes.php?division=dhaka
\`\`\`

## সিকিউরিটি নোটস

1. প্রোডাকশনে সমস্ত ডিফল্ট পাসওয়ার্ড পরিবর্তন করুন
2. `config/` ডিরেক্টরি লুকিয়ে রাখুন
3. নিয়মিত ডাটাবেস ব্যাকআপ নিন
4. SSL সার্টিফিকেট ইনস্টল করুন
5. ফায়ারওয়াল কনফিগার করুন
6. নিয়মিত সিকিউরিটি আপডেট করুন
